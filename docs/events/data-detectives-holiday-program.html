FAQ

Who can sign up?

This program is perfect for any high school-aged students with an interest in technology.

Do I need to know how to code?

No experience in coding is required! You just need to be comfortable with using a keyboard to type text. Our friendly tutors are there to help with everything you need to learn. If you are an experienced coder, we can give you a little bit extra to push your knowledge!

Do I need to bring a laptop?

If you have a Mac or Windows laptop you can use that for this program. If you don't, you can rent one from us for this course. Chromebooks do not work with the required software.

What will I learn?

We'll be coding in Python, to create programs for hardware that we can use to collect data and intercept secret message transmissions. The embedded-hardware tech we'll be coding with is called a Kookaberry. It's a bit like a micro:bit!

In addition to creating our own detective tools, we'll also learn how to uncover hidden patterns in data and graphs, and decrypt secret ciphered messages!
