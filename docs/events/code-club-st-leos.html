Register for St Leo's After-school Code Club

Registering is an expression of interest. A minimum number of registrations are needed for the code club to go ahead. Once that threshold is reached you will be contacted on the email provided regarding payment.

Since we're starting in Term 2, we're just doing single-term payment this term. At the start of future semesters we will offer a discount for enrolling for the entire semester instead of term by term.

Register once per student.

If you have any questions, email us at info@connectedcode.org or use the Contact Us page.
