Python Basics Cheat Sheet

Cut out the trickiest part of coding, remembering all that syntax! Give students the chance to let their ideas flow by putting these helpful reminders and examples at their finger tips.

This Python cheat sheet covers all of the essential syntax for your Aussie year 7–10 Digital Technologies Curriculum. It contains examples and reminders for:

  • Input and output
  • If statements
  • While and for loops
  • Functions
  • Lists
  • Dictionaries
  • Some helpful built-in functions
